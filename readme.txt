=== ZebraMap ===
Tags: time zones, map
Requires at least: 2.5
Tested up to: 3.2

Insert your ZebraMap time zone map with a shortcode.

== Description ==

Make a time zone map with your custom locations at <a href="http://zebramap.com">ZebraMap</a> and get the WordPress shortcode under the "Publish" tab. Install this plugin to make the shortcode work.

== Installation ==

1. Unzip "ZebraMap.zip" and upload the folder "zebramap" to the plugin directory.
2. Activate the plugin.
3. Make a time zone map with your custom locations at <a href="http://zebramap.com">ZebraMap</a>.
4. Copy the shortcode of the map and paste it into a page.

== Frequently Asked Questions ==

See under "Help" when configuring your map at <a href="http://zebramap.com">ZebraMap</a>
